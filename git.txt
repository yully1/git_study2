初始化仓库:  git init

设置全局用户名和邮箱: git config --global user.name "Your Name"
		  git config --global user.email "email@example.com"

生成公钥和私钥： ssh-keygen -t rsa -C "youremail@example.com"

新建一个本地分支: git checkout -b lee

查看所有的分支: git branch -a

关联远程仓库: git remote add origin git@github.com:magua1025/gitHome.git

push推送本地分支时,建立追踪关系: git push -u <远程主机名> <本地分支名>

增加已追踪的文件的修改删除及未追踪的文件提交到暂存区:  git add -A .

查看工作区状态: git status 

将暂存区里的改动给提交到本地的版本库，-m 是注释信息:  git commit -m 'message'

生成ssh key的公钥和私钥 需要回车三次  第一次是保存钥匙的位置,第二次是设置密码，第三次是确认密码

如果无法打开 github.com   修改 C:\Windows\System32\drivers\etc 路径下的 hosts文件 追加   20.205.243.166 github.com
这个ip地址是你在 https://tool.chinaz.com/dns/?type=1&host=github.com&ip= 这个链接输入github.com  查询得到的最小的ttl值的地址


cmd 输入  ipconfig/flushdns

1.本地把文件复制到git_home文件夹内
2.使用git add -A .命令把文件添加到暂存区
3.使用git commit -m "注释信息"  命令把文件提交到暂存区
4.使用 git branch -m main  把当前分支重命名 main 
5.中间可以穿插 git status 和 git branch -a 查看工作区状态和查看所有分支
6.本地分支关联 远程仓库 git remote add origin https://github.com/magua1025/git_study1.git
7.使用git push -u origin main 命令将暂存区所有内容推送到远端仓库,第一次使用时需要关联远端分支 main   下次提交时 直接git push 就可以了
